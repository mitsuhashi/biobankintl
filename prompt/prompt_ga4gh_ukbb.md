## GA4GH_UKBB対応表作成のためのプロンプト

GA4GH Data Access Agreement  (以下、GA4GH)とUKバイオバンクMTA（以下、UKBB）の条文対応表を作成してください。

### 入力ファイル

以下のファイルをプロンプトにアップロードします。

* GA4GH条文ファイル：mta_ga4gh_ja.md (https://raw.githubusercontent.com/mitsuhashi/biobankintl/refs/heads/main/mta/mta_ga4gh_ja.md)
* UKBB条文ファイル： mta_ukbb_ja.md (https://raw.githubusercontent.com/mitsuhashi/biobankintl/refs/heads/main/mta/mta_ukbb_ja.md)
* 第1回レビュー会議議論: 第1回レビュー会議議論.pdf


### 出力ファイル

| 列名 | 説明 | 値例 |
|---|---|---|
| トピック | GA4GH の条文タイトル（枝番単位でない） | 1. 定義、2. 利用目的 |
| GA4GH番号 | GA4GH の条文番号（枝番単位） | 1.1, 1.2, 2.1,... |
| GA4GH 条文 | GA4GH の条文（枝番単位の全文） | お客様が本契約の条項に違反していることを認識した場合、直ちにデータ提供者に通知しなければなりません。 |
| 判定 | GA4GHとUKBBの比較結果（同意：条文の内容が完全に同じ、混在：条文に同意点と相違点が混在している、相違：条文全体の内容が異なっている、GA4GHのみ：UKBBに対応する条文がない、UKBBのみ：GA4GHに対応する条文がない）| 「同意」、「混在」、「相違」、「GA4GHのみ」、「UKBBのみ」 |
| 同意点・相違点 | GA4GHとUKBBの同意点と相違点 | 【同意】 提供物（元データ/元試料）の権利は提供元に留保され、移転しない点。<br>【相違】GA4GHは明示的に「移転しない」と記述。UKBBは明文はないが、第8条で成果物の権利のみを乙に帰属させていることから、元試料の権利は甲にあると解釈される。 |
| UKBB番号 | UKBBの条文番号（枝番単位） | 2.1.2 |
| UKBB 条文 | UKBBの条文の対応箇所（前後の文言は割愛） | 2.1.2 承認された研究プロジェクトの許可された目的のためだけにのみ使用されること。 |
| 第1回レビュー会議議論 | 第1回レビュー会議で議論した内容 | 白石：兵器の利用禁止でいいのか。禁止兵器とした場合、禁止されてない兵器ならばよいのか。 |


### 出力ファイル作成時の注意点
* GA4GH の条文番号の順番に表示されます				
* 判定が「GA4GHのみ」の場合、UKBB番号およびUKBB条文は、「なし」となります。				
* 判定が「UKBBのみ」の場合、該当するUKBBの条文は最も近い内容のGA4GH条文の下に表示してください。その場合、GA4GH番号を表示し、GA4GH条文は、「なし」と表示してください。適当な挿入位置が見つからない場合は表の末尾に表示されます。
* 表の1行がGA4GHおよびUKBBの枝番単位の条文になります。
* 入力ファイルが読み込めていない場合は処理を停止してそのことを表示してください。あなたが既に持っている知識から回答しないでください。
* GA4GHおよびUKBBの条文は全て表示してください。
* 条文は ... などで省略せずに全部表示してください。
* 一度に出力できない場合は、分割して出力してください。一度に出力できる文書量の上限があるために内容を省略することはしないでください。
* 第1回レビュー会議議論の内容は、第1回レビュー会議議論.pdfから取得してください。UKBBの条文番号でマッチングしてください。UKBB条文番号と条文そのものも表示してください。
